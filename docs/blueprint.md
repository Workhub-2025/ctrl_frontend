# **App Name**: Emergency Services Assessment Platform

## Core Features:

- Secure Access: User authentication and role-based access control.
- Typing Speed Test: Typing test with one practice test and three real tests, each lasting one minute, measuring WPM and accuracy.
- Emergency Call Simulation: Simulated emergency call assessment with three pre-recorded calls, where candidates fill out a basic incident record with caller, location, and incident information.
- Situational Judgement Test: Situational judgment test featuring 15-20 multiple-choice questions to evaluate decision-making in emergency contexts.
- Performance Reporting: Report generation after each test to evaluate skills.
- Adaptive Testing: Use tool to analyze textual responses in real-time. Tool adjusts difficulty based on performance, leveraging past data and established rubrics for nuanced evaluation.

## Style Guidelines:

- Primary color: Deep navy blue (#1e3a8a) for authority and stability.
- Background color: Light-blue (#e0f2f7) for clarity and focus.
- Accent color: Teal (#0891b2) for a professional and calming effect.
- Body and headline font: 'Inter' sans-serif, for clear, accessible text in evaluations, and a balance of neutrality and legibility.
- Use icons that are clearly associated with emergency services to facilitate easy understanding, where appropriate.
- A clean, professional design will ensure an efficient evaluation workflow and avoid undue stress.
- Employ subtle animations to guide users and enhance the overall experience, while reducing distractions.
