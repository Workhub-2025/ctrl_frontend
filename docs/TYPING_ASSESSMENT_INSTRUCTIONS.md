# CTRL Typing Assessment

This assessment is designed to measure your typing speed and typing accuracy when reproducing written information. It evaluates your typing ability only; you are not expected to have any previous policing knowledge or control room experience.

## What You Will See

During the assessment, you will see:
* A passage of text
* A typing response box
* A countdown timer

## Your Task

You must type the passage exactly as displayed into the response box. Please pay attention to:
* Spelling
* Punctuation
* Capital letters
* Spacing

## Practice Exercise

There is one practice exercise available before you begin. This practice exercise is not scored and will not affect your final score.

## Live Assessment

* The live assessment contains one typing exercise.
* The assessment lasts 90 seconds.
* A timer will be visible throughout the exercise.
* The assessment will automatically end when the time expires.

## Step-by-Step Instructions

* **Step 1:** Read the passage carefully
* **Step 2:** Place your cursor in the response box
* **Step 3:** Begin typing when ready
* **Step 4:** Type the text as accurately as possible
* **Step 5:** Correct any mistakes if you notice them
* **Step 6:** Continue typing until the timer ends
* **Step 7:** Review your work if time remains
